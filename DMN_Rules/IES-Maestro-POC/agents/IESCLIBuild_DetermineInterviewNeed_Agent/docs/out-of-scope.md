# Out of Scope

This project is intentionally limited to deterministic advisory JSON generation.

- No Maestro implementation.
- No BPMN implementation.
- No DMN implementation.
- No BPMN file creation or modification.
- No coded action app.
- No real email.
- No real external service calls.
- No REST, SOAP, or SFTP integrations.
- No calls to ECM, IDR, ABLE, WMS, CNS, CMS, UIB, DOL, tax, BICS, or external systems.
- No authentication implementation.
- No eligibility approval or denial.
- No production policy engine.
